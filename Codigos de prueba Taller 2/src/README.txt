A continuacion se explican las instrucciones que se deben seguir para un correcto funcionamiento del program.

 - Instalacion de librerias

En el programa se hace uso de las librerias tkinter, pywswip, playsound y multiprocessing. Para evitar error,
se recomienda instalarlas en su maquina previo a ejecutar el programa.
Se instalan de la siguiente forma:

1. tkinter: pip install tkinter
2. pyswip: pip install pyswip
3. playsound: pip install playsound
4. multiprocessing: no requiere instalacion.

 - Ubicacion de los archivos

Se recomienda no mover de ubicacion los archivos que se encuentran dentro de la carpeta src, de lo contrario
se esperan errores en el funcionamiento.

 - Modo de uso

Una vez descargado el proyecto se debe extraer e ingresar a la carpeta src, una vez dentro de la carpeta del proyecto
se debe abrir el archivo main.py en su IDLE favorito. 
Una vez abierto el codigo fuente para ejecutarlo se debe presionar el boton ejecutar y a continuacion podra observar la ventana de bienvenida.

 - Interaccion con la interfaz

La primera ventana que podrá observar al ingresar al sistema es la de bienvenida que le va a solicitar su nombre, se recomienda ingresarlo, ya que,
de lo contrario se enviará una alerta. Cuando ingrese su nombre presione el botón "Ingresar"

Luego de presionar el botón "Ingresar" se abrirá la ventana de selección de preferencias donde usted podrá ingresar aquellas preferencias
que encuentre necesarias para consultar al sistema. Cabe destacar que la opcion de nivel de habilidad es obligatorio seleccionarlo.
Una vez haya seleccionado sus preferencias debe presionar el botón consultar que abrira una nueva ventana con un tabla que contiene la lista de juegos
que se adecuan a sus preferencias.
